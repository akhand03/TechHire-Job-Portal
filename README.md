# TechHire – Job Portal Web Application

A full-stack job portal web application that connects Job Seekers, Employers, and Admins on a single platform.

## Features
- Role-based system for Job Seekers, Employers, and Admins
- Secure authentication using JWT and bcrypt password hashing
- Job posting and management for employers
- Dynamic job search with keyword, location, and job type filters
- Resume upload with file type and size validation (Multer)
- Application tracking system

## Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Authentication:** JWT (JSON Web Token), bcrypt
- **File Upload:** Multer
- **Other:** dotenv, cors
